# Anemia-Regression-Analysis
Healthcare analysis project designed to inquire which factors lead to which anemia diagnosis.
    
    Project Overview
This project is investigating the classification of multiple anemia types using machine learning and statistical modeling techniques. The objective is to identify relationships between hematological blood results and the various diagnosis associated.

    Research Question
Which predictors are associated with different anemia types?

    Anemia Classification Dataset
This dataset contains laboratory blood results and classification for healthy individuals and multiple different anemia types.

    Variables analyzed
Hemoglobin (HGB)
Red Blood Cell Count (RBC)
Mean Corpuscular Volume (MCV)

    Target Variable
Diagnosis
    Healthy (referance category)
    Multiple Anemia Types

    Methadology
These analytical procedures were preformed:
Data cleaning and processing
Exploratory Data Analysis (EDA)
Descriptive statistics
Class distribution analysis
Multinominal evaluation and interpretation
Statistical significance testing of predictors

    Software Used
R

    Results
Several blood indicators were statistically significant predictors of anemia classification
The multinominal logisitcal regression model successfully differentiated healthy patients from various anemia types
Blood values demonstrated a strong predictive value to identifying anemia diagnosis
